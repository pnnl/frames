In this folder are tests I have run on the ATO. They are, to my best intentions, an accurate testing of the necessary functionality of ATO merging.

Also please make note that I used my intuition on these tests, and no systematic process to assure that every single value was merged/converted correctly.

Most tests, if not all, have an ato, csv, and xls file. To open and run each test, use the atotest.gid. you must rename the appropriate test you want to run to atotest.ato. for example, if you want to test cartesian to polar with cartToPolar, then rename the cartToPolar.ato to atotest.ato. Then you can open up the atotest.gid with FRAMES.

ato -- The input ato data that I have created by hand or generated using the Mepas 4.1 air model, according to the specifications listed on the FRAMES 1.x website.
csv -- the result of the ato merge.
xls -- a spreadsheet I was using to compare and/or graph results.

cartToPolar.ato -- used to test conversion from cartesian to polar.
difftypes.ato -- used to test merging of one type cartesian with one type polar.
interpolated.ato -- testing that the actual values of each grid cell were correctly interpolated.
offset.ato -- testing that grids from one dataset will properly be merged with grids from a different dataset that is offset, and that the resulting grids contain only the merge of overlapping points, and to resize the output grids to encompass the bounds of both input dataset grids.
polarToCart.ato -- that values will be correctly converted from polar to cartesian.
