5.27 DIRECT RADIATION

    The direct radiation exposure pathway involves external radiation exposure for a period of time to a measures radiation field. The two parameters for this pathway are the daily exposure time and the effective rate constant for reduction of the measured direct radiation dose rate. The default values for these parameters are given in Table 5.26.

    The daily exposure time default values represent continuous exposure. The default reduction rate constant represents a radiation field that is constant with time.
