5.25 SOIL EXTERNAL


    External exposure to contaminated soil is considered for the atmospheric transport pathway and the measured soil pathway. Default parameter values for these exposure pathways are given in Table 5.24.



5.26 AIR EXTERNAL

    Exposure to air contaminated with radioactive material may result in external exposure while an individual is near the plume. This exposure pathway requires definition of the daily exposure time, as given in Table 5.25.

    The default value for the air external exposure time is set to 24 hours representing continuous exposure.
