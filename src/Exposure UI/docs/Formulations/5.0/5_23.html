5.23 BOATING EXTERNAL
    The boating external exposure pathway is the same as the swimming external exposure model, except a shielding correction is applied for less than complete immersion in the water. Default parameter values for the boating external exposure pathway are given in Table 5.22.

    Note that the product of boating event time and swimming frequency is equal to an average daily swimming time of 0.033 hours, consistent with the initial MEPAS daily boating exposure time.
