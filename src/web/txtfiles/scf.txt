<h2>Soil Concentration File (SCF)</h2>
There are currently four qualified types of the SCF, those qualifiers are: <font color="#0000FF">"Soil-Total", "Soil-Dissolved" "Sediment-Total", "Sediment-Dissolved"</font>. The constituent concentrations are a spatial average throughout the volume of the soil/sediment zone. The concentrations units in the zone depend on whether the constituent is a radionuclide or chemical and which qualifiers is used and constituent phase measured.  These concentration values are the instantaneous values at that specified time, not an average. Every data set denotes another location, in most cases a downstream icon, but not always so. There are NO limits on number of data sets, constituents, progeny, and time/concentration pairs.
<p>
The following outline is a description of the basic information provided in an SCF. An example SCF is included after the outline. Comparing the line by line description in the outline to the example should clarify and help locate data contained in the SCF. Lines beginning with ';' are descriptive comments only and will not appear in the actual SCF.

<br>&nbsp;&nbsp;&nbsp;&nbsp;Number of Lines in section - <B>[INTEGER]</B>
<br>&nbsp;&nbsp;<B><I>End Line</I></B>
<br>&nbsp;&nbsp;<B><I>A line with:</I></B>

<br>&nbsp;&nbsp;<B><I>For each</I></B> 'Header Line'
<br>&nbsp;&nbsp;&nbsp;&nbsp;<B><I>A line with:</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Header information - <B>[STRING]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;<B><I>End Line</I></B>
<br>&nbsp;&nbsp;<B><I>Next</I></B> 'Header Line'
<br>&nbsp;&nbsp;<B><I>A line with:</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;Number of Data Sets - <B>[INTEGER]</B>
<br>&nbsp;&nbsp;<B><I>End Line</I></B>
<br>&nbsp;&nbsp;<B><I>For each</I></B> 'Data Set'
<br>&nbsp;&nbsp;&nbsp;&nbsp;<B><I>A line with:</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Data Set name - <B>[STRING]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;This is the location name or ID.
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;<B><I>If</I></B> the set is intended for a particular consumming module's use <B><I>then</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;&nbsp;&nbsp;'Data Set Name' = Consuming Module ID
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;<B><I>ElseIf</I></B> the set is intended for any and all consuming modules <B><I>then</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;&nbsp;&nbsp;'Data Set Name' = "All" or user may be module defined
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;&nbsp;&nbsp;'Number of Data Sets' = 1
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;<B><I>EndIf</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Valid SCF Qualifier for this data set - <B>[STRING]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;<font color="#0000FF">"Soil-Total" / "Soil-Dissolved" / "Sediment-Total / "Sediment-Dissolved" </font>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;x Dimension of volume - <B>[FLOAT]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;x Dimension unit - <B>[Const STRING]</B> - "m"
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;y Dimension of volume - <B>[FLOAT]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;y Dimension unit - <B>[Const STRING]</B> - "m"
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;z Dimension of volume - <B>[FLOAT]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;z Dimension unit - <B>[Const STRING]</B> - "m"
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Number of Constituents - <B>[INTEGER]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Centroid Easting - <B>[FLOAT]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Easting unit - <B>[Const STRING]</B> - "m"
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Centroid Northing - <B>[FLOAT]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Northing unit - <B>[Const STRING]</B> - "m"
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Centroid Depth below ground level - <B>[FLOAT]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Depth unit - <B>[Const STRING]</B> - "m"
<br>&nbsp;&nbsp;&nbsp;&nbsp;<B><I>End Line</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;<B><I>For each</I></B> 'Constituent'
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>A line with:</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Constituent Name - <B>[STRING]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Constituent ID - <B>[STRING]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Time unit - <B>[Const STRING]</B> - "yr"
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Concentration unit - <B>[Const STRING]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;File qualifier determines concentration units and aids in connection scheme:
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;Soil-Total - "pCi/kg" / "mg/kg" - Total constituent mass (soil water free product) per dry weight of soil
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;Soil-Dissolved - "pCi/L / mg/L" - Dissolved constituent mass per volume of soil water
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;Sediment-Total - "pCi/kg" / "mg/kg" - Total constituent mass (sediment water free product) per dry weight of sediment
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;Sediment-Dissolved - "pCi/L" / mg/L - Dissolved mass in water per volume of water
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Number of Time Concentration Pairs - <B>[INTEGER]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Number of progeny - <B>[const INTEGER]</B> - 0
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;&nbsp;&nbsp;must be set to zero for 1.6 spec
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>End Line</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>For each</I></B> 'Time Concentration Pair'
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>A line with:</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Time - <B>[FLOAT]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Constituent Concentration - <B>[FLOAT]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>End Line</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>Next</I></B> 'Time Concentration Pair'
<br>&nbsp;&nbsp;&nbsp;&nbsp;<B><I>Next</I></B> 'Constituent'
<br>&nbsp;&nbsp;<B><I>Next</I></B> 'Data Set'
<br><B><I>Next</I></B> 'Module'
