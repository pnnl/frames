<h2>Body Burden File (BBF)</h2>
The Body Burden File (BBF) contains a time series of chemical body burden or radiological dose (i.e., dose/body burden versus time). The concentrations are in pCi/kg or mg/kg, depending on whether the constituent is a radionuclide or a chemical. These concentration values are the instantaneous values at that point in time, not an average. There are NO limits on the number of organisms, constituents, or concentrations.
<p>
The following outline is a description of the basic information provided in the BBF. An example BBF is included after the outline. Comparing the line-by-line description in the outline to the example should clarify and help locate data contained in the BBF. Lines beginning with ';' are descriptive comments only and will not appear in the actual BBF.
<br>
<br><B><I>For each</I></B> 'Module' producing a BBF section
<br>&nbsp;&nbsp;<B><I>A line with:</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;Module Name - <B>[STRING]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;Number of Lines in section - <B>[INTEGER]</B>
<br>&nbsp;&nbsp;<B><I>End Line</I></B>
<br>&nbsp;&nbsp;<B><I>A line with:</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;Number of Header Lines - <B>[INTEGER]</B>
<br>&nbsp;&nbsp;<B><I>End Line</I></B>
<br>&nbsp;&nbsp;<B><I>For each</I></B> 'Header Line'
<br>&nbsp;&nbsp;&nbsp;&nbsp;<B><I>A line with:</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Header information - <B>[STRING]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;<B><I>End Line</I></B>
<br>&nbsp;&nbsp;<B><I>Next</I></B> 'Header Line'
<br>&nbsp;&nbsp;<B><I>A line with:</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;Number of Data Sets - <B>[INTEGER]</B>
<br>&nbsp;&nbsp;<B><I>End Line</I></B>
<br>&nbsp;&nbsp;<B><I>For each</I></B> 'Data Set'
<br>&nbsp;&nbsp;&nbsp;&nbsp;<B><I>A line with:</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;File Extension - <B>[STRING]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;<B><I>If</I></B> 'Data Set' is user defined <B><I>then</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;&nbsp;&nbsp;'File Extension' is ""
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;<B><I>Else</I></B> 'File Extension' is a valid PDCF Extension (i.e. "SCF"/ "WCF" / ...)
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;&nbsp;&nbsp;The consumed PDCF's extension used to generate this data set
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;<B><I>EndIf</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;<B><I>If</I></B> more than one input file was used to generate this data set <B><I>then</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;&nbsp;&nbsp;Separate each with a semicolon (e.g. "SCF:WCF")
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;<B><I>EndIf</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;File Qualifier - <B>[STRING]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;<B><I>If</I></B> 'Data Set' is user defined <B><I>then</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;&nbsp;&nbsp;'File Qualifier' is the media that exposed this data set
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;<B><I>Else</I></B> 'File Qualifier' is a valid PDCF Qualifier for 'File Extension' (i.e. "Sediment" / "Surface Water" / ...)
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;&nbsp;&nbsp;The consumed PDCF's qualifier used to generate this data set
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;<B><I>EndIf</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;<B><I>If</I></B> more than one input file was used to generate this data set <B><I>then</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;&nbsp;&nbsp;Separate each with a semicolon (e.g. "Sediment:Surface Water")
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;<B><I>EndIf</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Number of Organisms - <B>[INTEGER]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Number of Variability Levels - <B>[INTEGER]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;if discrete set to 1
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;else 1 or more are required for crosstab
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Number of Uncertainty Levels - <B>[INTEGER]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;if discrete set to 1
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;else 2 or more are required for crosstab
<br>&nbsp;&nbsp;&nbsp;&nbsp;<B><I>End Line</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;<B><I>A line with:</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>For each</I></B> 'Variability Level'
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Variability Level label - <B>[STRING]</B> - (Discreet, 10%, 90%...)
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>Next</I></B> 'Variability Level'
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>For each</I></B> 'Uncertainty Level'
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Uncertainty Level label - <B>[STRING]</B> - (Discreet, 5%, 50%, 95%...)
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>Next</I></B> 'Uncertainty Level'
<br>&nbsp;&nbsp;&nbsp;&nbsp;<B><I>End Line</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;<B><I>For each</I></B> 'Organism'
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>A line with:</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Organism Name - <B>[STRING]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Number of Constituents - <B>[INTEGER]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>End Line</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>For each</I></B> 'Constituent'
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>A line with:</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Constituent Name - <B>[STRING]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Constituent ID - <B>[STRING]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Time units - <B>[Const STRING]</B> - "yr"
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Concentration units - <B>[Const STRING]</B> - "mg/kg"
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Number of Time Concentrations pairs - <B>[INTEGER]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Number of Progeny - <B>[INTEGER]</B> - 0
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;&nbsp;&nbsp;must be set to zero for 1.6 spec
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>End Line</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>For each</I></B> 'Time Concentration Pair'
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>A line with:</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Time - <B>[FLOAT]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>For each</I></B> 'Variability level'
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>For each</I></B> 'Uncertainty Level'
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Concentration value - <B>[FLOAT]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>Next</I></B> 'Uncertainty Level'
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>Next</I></B> 'Variability level'
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>End Line</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>Next</I></B> 'Time Concentration Pair'
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>Next</I></B> 'Constituent'
<br>&nbsp;&nbsp;&nbsp;&nbsp;<B><I>Next</I></B> 'Organism'
<br>&nbsp;&nbsp;<B><I>Next</I></B> 'Data Set'
<br><B><I>Next</I></B> 'Module'
