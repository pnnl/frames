<h2>Exposure Pathway File (EPF)</h2>
The output from the Exposure Pathways component is the concentration of each pollutant for a given route and pathway. The EPF is the input to the Receptor Intake Component which produces estimates of pollutant intakes.
<p>
The following outline is a description of the basic information provided in the EPF. An example EPF is included after the outline. Comparing the line by line description in the outline to the example should clarify and help locate data contained in the EPF. Lines beginning with ';' are descriptive comments only and will not appear in the actual EPF.
<br>
<br><B><I>For each</I></B> 'Module' producing an EPF section
<br>&nbsp;&nbsp;<B><I>A line with:</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;Module Name - <B>[STRING]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;Number of Lines in section - <B>[INTEGER]</B>
<br>&nbsp;&nbsp;<B><I>End Line</I></B>
<br>&nbsp;&nbsp;<B><I>A line with:</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;Number of Header Lines - <B>[INTEGER]</B>
<br>&nbsp;&nbsp;<B><I>End Line</I></B>
<br>&nbsp;&nbsp;<B><I>For each</I></B> 'Header Line' <B><I>A line with:</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;<B><I>A line with:</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Header information - <B>[STRING]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;<B><I>End Line</I></B>
<br>&nbsp;&nbsp;<B><I>Next</I></B> 'Header Line'
<br>&nbsp;&nbsp;<B><I>A line with:</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;Number of Data Sets - <B>[INTEGER]</B>
<br>&nbsp;&nbsp;<B><I>End Line</I></B>
<br>&nbsp;&nbsp;<B><I>For each</I></B> 'Data Set'
<br>&nbsp;&nbsp;&nbsp;&nbsp;<B><I>A line with:</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Data set type - <B>[Const STRING]</B> - "acute" / "chronic"
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;File Extension - <B>[STRING]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;<B><I>If</I></B> 'Data Set' is user defined <B><I>then</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;&nbsp;&nbsp;'File Extension' is ""
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;<B><I>Else</I></B> 'File Extension' is a valid PDCF Extension (i.e. "SCF"/ "WCF" / ...)
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;&nbsp;&nbsp;The consumed PDCF's extension used to generate this data set
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;<B><I>EndIf</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;<B><I>If</I></B> more than one input file was used to generate this data set <B><I>then</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;&nbsp;&nbsp;Separate each with a semicolon (e.g. "SCF:WCF")
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;<B><I>EndIf</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;File Qualifier - <B>[STRING]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;<B><I>If</I></B> 'Data Set' is user defined <B><I>then</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;&nbsp;&nbsp;'File Qualifier' is the media that exposed this data set
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;<B><I>Else</I></B> 'File Qualifier' is a valid PDCF Qualifier for 'File Extension' (i.e. "Sediment" / "Aquifer" / ...)
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;&nbsp;&nbsp;The consumed PDCF's qualifier used to generate this data set
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;<B><I>EndIf</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;<B><I>If</I></B> more than one input file was used to generate this data set <B><I>then</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;&nbsp;&nbsp;Separate each with a semicolon (e.g. "Sediment:quifer")
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;<B><I>EndIf</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Number of Exposure Media Points - <B>[INTEGER]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Number of Constituents - <B>[INTEGER]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;<B><I>End Line</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;<B><I>For each</I></B> 'Exposure Media Point'
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>A line with:</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;X coordinate - <B>[FLOAT]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;X coordinate unit - <B>[Const STRING]</B> - "km"
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Y coordinate - <B>[FLOAT]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Y coordinate unit - <B>[Const STRING]</B> - "km"
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>End Line</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;<B><I>Next</I></B> 'Exposure Media Point'
<br>&nbsp;&nbsp;&nbsp;&nbsp;<B><I>For each</I></B> 'Constituent'
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>A line with:</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Constituent Name - <B>[STRING]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Constituent ID - <B>[STRING]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Number of Progeny - <B>[INTEGER]</B> - 0
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;&nbsp;&nbsp;must be set to zero for 1.6 spec
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Number of Exposure Start Times - <B>[INTEGER]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>End Line</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>For each</I></B> 'Exposure Start Time'
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>A line with:</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Exposure Start Time - <B>[FLOAT]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Exposure Start Time unit - <B>[Const STRING]</B> - "yr"
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Exposure Duration - <B>[FLOAT]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Exposure Duration unit - <B>[Const STRING]</B> - "yr"
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Number of Exposure Pathways - <B>[INTEGER]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>End Line</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>For each</I></B> 'Exposure Pathway'
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>A line with:</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Exposure Pathway Name - <B>[STRING]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;"Air" / "Indoor air" / "Ground" / "Leafy vegetables" /
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;"Root vegetables" / "Fruit" / "Grain" / "Beef" /
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;"Poultry" / "Milk" / "Eggs" / "Soil" / "Swimming" /
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;"Boating" / "Shoreline" / "Water" / "Fish" / "Mollusks" /
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;"Crustacea" / "Aquatic plants" / "Showering" / ...
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Exposure Route - <B>[STRING]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;"Ingestion" / "Inhalation" / "Dermal" / "External"
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Exposure Concentration unit - <B>[Const STRING]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;"Bq/kg" / "Bq/l" / "Bq/m3" for radionuclides
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;"mg/kg" / "mg/l" / "mg/m3" for non-radionuclides
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;"Sv" for external routes
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>End Line</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>A line with:</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Exposure Concentration - <B>[Set of FLOAT]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;a concentration <B><I>For each</I></B> 'Exposure Media Point'
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>End Line</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>Next</I></B> 'Exposure Pathway'
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>Next</I></B> 'Exposure Start Time'
<br>&nbsp;&nbsp;&nbsp;&nbsp;<B><I>Next</I></B> 'Constituent'
<br>&nbsp;&nbsp;<B><I>Next</I></B> 'Data Set'
<br><B><I>Next</I></B> 'Module'
