<h2>Ecological Effects File (EXF)</h2>
There are three types of output contained in the EXF file, Ecological effects, Ecological Hazard Quotients (EHQs) and Organism Intake Quotients (OIQs).  All types of output are expected to be placed in the same comma-separated file with an EXF extension.  A module section can contain one or all types in any order, but can only contain one of each type.
<p>
The following outline is a description of the basic information provided in an EXF. An example EXF file is included after the outline. Comparing the line by line description in the outline to the example should clarify and help locate data contained in the EXF file. Lines beginning with ';' are descriptive comments only and will not appear in the actual EXF.
<h3>Ecological Effects</h3>
This output type describes the effects based on an acute duration, continuous criterion and effects curve.  The data set also provides the probability of equaling or exceeding a concentration based on the input concentration curve.
<br>
<br><B><I>For each</I></B> 'Module' producing an EXF section
<br>&nbsp;&nbsp;<B><I>A line with:</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;Module Name - <B>[STRING]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;Number of Lines in section - <B>[INTEGER]</B>
<br>&nbsp;&nbsp;<B><I>End Line</I></B>
<br>&nbsp;&nbsp;<B><I>A line with:</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;Effects label - <B>[Const STRING]</B> - "Effects"
<br>&nbsp;&nbsp;&nbsp;&nbsp;Number of Organisms - <B>[INTEGER]</B>
<br>&nbsp;&nbsp;<B><I>End Line</I></B>
<br>&nbsp;&nbsp;<B><I>For each</I></B> 'Organism'
<br>&nbsp;&nbsp;&nbsp;&nbsp;<B><I>A line with</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Organism Common Name - <B>[STRING]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Organism Scientific Name - <B>[STRING]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Number of Constituents - <B>[INTEGER]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;<B><I>End Line</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;<B><I>For each</I></B> 'Constituent'
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>A line with</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Constituent Name - <B>[STRING]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Constituent CASID - <B>[STRING]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Number of Effect Regions - <B>[INTEGER]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>End Line</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>For each</I></B> ‘Effect Region'
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>A line with:</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Percent of Time Organism Exceeding Effect - <B>[FLOAT]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Effect Description - <B>[STRING]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>End Line</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>Next</I></B> ‘Effect Region’
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>A line with:</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;1st Column Label - <B>[Const STRING]</B> - "Concentration in g/ml"
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;2nd Column Label - <B>[Const STRING]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;;"Probability of Equaling or Exceeding a Concentration, based on Exposure Duration (%)"
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Number of Probabilities - <B>[INTEGER]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>End Line</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>For each</B></I> ‘Probability'
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>A line with:</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Concentration - <B>[FLOAT]</B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Probability of Equaling or Exceeding a Concentration
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>End Line</I></B>
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<B><I>Next</I></B> ‘Probability’
<br>&nbsp;&nbsp;&nbsp;&nbsp;<B><I>Next</I></B> ‘Constituent’
<br>&nbsp;&nbsp;<B><I>Next</I></B> ‘Organism’
<br><B><I>Next</I></B> 'Module'
