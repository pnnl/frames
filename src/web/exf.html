Ecological Effects File (EXF)

There are two types of output contained in the EXF file, Ecological effects and Ecological Hazard Quotients (EHQs). Both types of output are expected to be placed in the same comma-separated file with an EXF extension. A module section can contain one or both types in any order, but can only contain one of each type.

The following outline is a description of the basic information provided in an EXF. An example EXF file is included after the outline. Comparing the line by line description in the outline to the example should clarify and help locate data contained in the EXF file. Lines beginning with ';' are descriptive comments only and will not appear in the actual EXF.

Ecological Effects

This output type describes the effects based on an acute duration, continuous criterion and effects curve. The data set also provides the probability of equaling or exceeding a concentration based on the input concentration curve.

For each 'Module' producing an EXF section
  A line with:
    Module Name - [STRING]
    Number of Lines in section - [INTEGER]
  End Line
  A line with:
    Effects label - [Const STRING] - "Effects"
  End Line
  For each 'Organism'
    ;The count is assumed to be the number of Organisms being assessed
    A line with
      Organism Common Name - [STRING]
      Organism Scientific Name - [STRING]
    End Line
    A line with:
      Constituent Name - [STRING]
      Constituent CASID - [STRING]
    End Line
    For each ‘Effect Region’
      A line with:
        Percent time at given Effect Region - [FLOAT]
        Effect Region description - [STRING]
      End Line
    Next ‘Effect Region’
    A line with:
      1st Column Label - [Const STRING] - "Concentration in g/ml"
      2nd Column Label - [Const STRING]
        ;"Probability of Equaling or Exceeding a Concentration, based on Exposure Duration (%)"
    End Line
    For ‘I’ = 1 to 11
      A line with:
        Concentration - [FLOAT]
        Probability of Equaling or Exceeding a Concentration
      End Line
    Next ‘I’
  Next ‘Organism’
Next 'Module'

Ecological Hazard Quotients

This output type describes the EHQ over time for a given duration and toxicity reference dose, the probability of equaling or exceeding an EHQ for a given duration and toxicity reference dose, and the probability of equaling or exceeding a body burden.

For each 'Module' producing an EXF section
  A line with:
    Module Name - [STRING]
    Number of Lines that follow for this module's section
  End Line
  A line with:
    EHQ label - [Const STRING] - "EHQs"
  End Line
  For each 'Organism'
    For each 'Constituent'
      For each 'Effect Level'
        For each 'Body Part of Concern'
          For each 'Type of Effect'
            For each 'Concentration'
              A line with:
                Description of output
              End Line
              A line with:
                1st Column Label - [Const STRING] - "EHQ"
                1st Column Unit - [Const STRING] - ""
                2nd Column Label - [Const STRING] - "Time"
                2nd Column Unit - [Const STRING] - "yr"
              End Line
              A line with:
                Number of Time Periods
              End Line
              For each ‘Time Period’ A line with:
                A line with:
                  EHQ - [FLOAT]
                  Time - [FLOAT]
                End Line
              Next ‘Time Period’
              A line with:
                1st Column Label - [Const STRING] - "EHQ"
                1st Column Unit - [Const STRING] - ""
                2nd Column Label - [Const STRING] - "Probability of Equaling or Exceeding EHQ"
                2nd Column Unit - [Const STRING] - "%"
              End Line
              A line with:
                Number of Time Periods
              End Line
              For each ‘Time Period’
                A line with:
                  EHQ - [FLOAT]
                  Probability of Equaling or Exceeding EHQ - [FLOAT]
                End Line
              Next ‘Time Period’
              A line with:
                1st Column Label - [Const STRING] - "Body Burden"
                1st Column Unit - [Const STRING] - "mg/kg"
                2nd Column Label - [Const STRING] - "Probability of Equaling or Exceeding Body burden"
                2nd Column Unit - [Const STRING] - "%"
              End Line
              A line with:
                Number of Time Periods
              End Line
              For each ‘Time Period’
                A line with:
                  Body burden - [FLOAT]
                  Probability of Equaling or Exceeding Body burden - [FLOAT]
                End Line
              Next ‘Time Period’
            Next ‘Concentration’
          Next ‘Type of Effect’
        Next ‘Body Part of Concern’
      Next ‘Effect Level’
    Next ‘Constituent’
  Next ‘Organism’
Next 'Module'
