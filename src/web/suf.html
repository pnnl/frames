Sensitivity/Uncertainty File (SUF)

The following outline is a description of basic information in an SUF. An example SUF file is included after outline. Comparing line by line description in outline to example should clarify and help locate data contained in SUF file. Lines beginning with ';' are descriptive comments only and will not appear in the actual SUF.

For each 'Module' producing an SUF section
  A line with:
    Module Name - [STRING]
    Number of Lines in section - [INTEGER]
  End Line
  A line with:
    Number of Header Lines - [INTEGER]
  End Line
  For each 'Header Line'
    A line with:
      Header information - [STRING]
    End Line
  Next 'Header Line'
  A line with:
    Number of Input (Modified) Variables - [INTEGER]
    Number of Output (Monitored) Variables - [INTEGER]
  End Line
  For each 'Input Variable'
    A line with:
      Variable alias - [STRING]
      Variable description - [STRING]
    End Line
  Next 'Input Variable'
  For each 'Output Variable'
    A line with:
      Variable alias - [STRING]
      Variable description - [STRING]
    End Line
  Next 'Input Variable'
  A line with:
    Number of Realizations - [INTEGER]
  End Line
  A line with:
    Realization
    For each 'Input Variable'
      Variable alias - [STRING]
    Next 'Input Variable'
    For each 'Output Variable'
      Variable alias - [STRING]
    Next 'Input Variable'
  End Line
  For each 'Realization'
    A line with:
      Realization index
      For each 'Input Variable'
        Variable value used - [STRING]
      Next 'Input Variable'
      For each 'Output Variable'
        Variable value monitored - [STRING]
      Next 'Input Variable'
    End Line
  Next 'Realization'
Next 'Module'
