Graphical Viewer Help

Graphical viewers show the response curve produced (e.g. concentration, dose) given the criteria selections made by the user.

Menu Bar:
  The menu bar spans the top portion of the viewer window and provides the user with several options.

  • Chart: Opens Excel, creating workbooks and charts.
  • Print: Saves a Tagged Image Format (*.tif) file with a snapshot of the dataset.
  • Save: Saves a Tab Delimited (*.txt) file with the values from the dataset.
  • Help: Contains the 'About' and 'How To...' information. The 'About' selection displays the description file of the viewer.

Benchmarks File: (If availble)
  This gives the user the ability to specify a TREECSBenchmarkInfo XML file containing pathway benchmarks for constituents. If applicable values are found in the file they will appear as horizontal lines on the chart. Values may be omitted from the chart if they exceed the maximum value of the concentration curve and/or confidence intervals by more than 10%.

Number of Charts:
  Allows user to make more than one chart at a time, a maximum of 25 charts are possible.

Report Details:
  The user can selection a value for each criteria. Some or all of the criteria selections may be present depending on the file type being viewed.
  Note: Criteria selections are to be made in descending order as each selection may affect the criteria selections after it.

  • Dataset:   User can choose the dataset.
  • Location:   User can choose the location.
  • Age:   User can choose the age group.
  • Constituent:   User can choose a single constituent, 'All Chemicals', 'All Radionuclides' or 'All Constituents'.
  • Measure:   User can choose the measure depending upon what is applicable to the file type.
  • Unit:   User can choose the unit depending upon what is applicable to the file type.
  • Route:   User can choose a single route of exposure or 'All'.
  • Pathway:   User can choose a single pathway of exposure or 'All'.
  • Organ:   User can choose a single organ or 'All'.
