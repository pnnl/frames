Automated Test for CSTRM10b

PURPOSE

This case verifies that the CSTRM correctly calculates volatilization based on varying site data. Four scenarios were set up in this test case. Scenario 10b, was run with no NAPL and no clean layer. The constituent benzene was selected for this case.

REQUIREMENTS

  • Produce time-varying contaminant mass fluxes for air, water, and soil deposition (from air) from the source zone that meet the data specifications of the FRAMES software system.
  • Compute contaminant mass fluxes for all specified loss routes or allow the user to input known contaminant mass fluxes for some loss routes and compute the contaminant mass fluxes for the remaining specified loss routes.
  • Address chemical and radioactive constituents.
