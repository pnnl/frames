Automated Test for CSTRM09

PURPOSE

This case verifies that the CSTRM correctly calculates suspension based on varying site data. The scenario was run with wind and mechanical erosion in the limited erosion scenario. The constituent Chromium VI was selected for this case. In addition, the on-line help option will be checked in this test case. This will be done by selecting several parameters and clicking on help to see if a help page describing the parameters appears.

REQUIREMENTS

  • Provide on-line help in an HTML format that provides the user with descriptions of all input parameters required by the MUI.
  • Compute wind, mechanical, and overland soil erosion rates.
