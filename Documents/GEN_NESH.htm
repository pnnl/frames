GENII NESHAPS Report Generator User Interface

Method Selections

Three options are available.  Check the appropriate box

Include Atmospheric Dispersion and Deposition Estimates:  This output provides the minimum required for reporting doses and risks to the Maximally Exposed Individual.  This option will report the Name and Address of the facility being modeled along with the  doses and/or risks to individuals.

Selection of this option activates a "Site/User Identification" tab.  Several areas are provided for the input of identifying text information.   This information is repeated verbatim in the output.

Include Population Dose/Risk Estimates:  This option allows calculation of doses and/or risks of the population within the boundaries of the calculation grid.  The Population input file structure must match that of the calculation grid - e.g., the number of radial directions and distances must be those used in the atmospheric dispersion module.  The current version of the module does not have a "Browse" feature, so the exact filename (and path, if the file is not stored in the FRAMES directory) must be entered.

Selection of this option also activates the "Site/User Identification" tab.  Several areas are provided for the input of identifying text information.   This information is repeated verbatim in the output.
 

Provide only Individual Dose/Risk Output:  This option allows printing of a short summary of the individual output without any identifying information attached.  No additional method or parameter information is required.
