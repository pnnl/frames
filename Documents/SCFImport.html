SCF Spreadsheet Import Help

The SCF Spreadsheet Import module allows the user to import constituent concentration and soil property data from a spreadsheet template. The template file is named "SCFImportTemplate.xls" and is located in the FRAMES directory. The spreadsheet consists of two worksheets labeled "Chemicals" and "Soil" as shown in Figure 1. The constituent concentration data is entered on the "Chemicals" worksheet and the soil property data is entered on the "Soil" worksheet.

Figure 1.  SCF Spreadsheet Import Template

Concentration data can be entered for multiple sites by changing the number of locations in cell B8 (highlighted in yellow) and then appending the appropriate rows at the bottom of the worksheet for each new location. Each location data set must start with a data set header row (as shown in row 9) and is then followed by a row of site specific column headers and a row containing the site-specific data such as the site name, site coordinates in the x, y, and z planes, and the number of parent constituents. The next row (as shown in row 12) is a constituent data header row. The data for the constituents is then entered consecutively below this row and each constituent data set must contain the appropriate number of time/concentration pairs as specified in column D. If progeny data is required for a constituent, enter the number of progeny in column C for the specified constituent and then enter the progeny data directly after the parent constituent as shown in Figure 2 for STRONTIUM-90.

Figure 2.  SCF Spreadsheet Import Template with Progeny Data

Soil property data is entered for each site in a similar fashion on the "Soil" worksheet as shown in Figure 3. Note that row 12 is a soil property data header row (similar to the constituent data header row on the Chemicals worksheet) and the property data follows for each property identified. Each property must have the appropriate number of time/property pairs as shown in column C.

Figure 3.  SCF Spreadsheet Import Template Soil Property Data

After entering the spreadsheet template data, launch the SCF Spreadsheet Import module interface. The module interface main screen is shown in Figure 4.

Figure 4.  SCF Spreadsheet Import Module Main Screen

To load the spreadsheet template click the "Browse" button and select the appropriate spreadsheet file as shown in Figure 5.

Figure 5.  Using the Open File Dialog to Load the Spreadsheet Template

When the spreadsheet has been loaded the screen will appear as shown in Figure 6. The file loaded will be displayed in the "Spreadsheet Filename" box and "Chemicals" will be shown in the "Region (worksheet)"  box.

Figure 6.  SCF Spreadsheet Import Module with Template Loaded

Click the down arrow on the "Location" dropdown box to select the appropriate location data to use for this module. Once a site has been chosen the constituent name, CAS ID, and the beginning date for the time/concentration pairs will appear in the list at the bottom of the screen as shown in Figure 7.

Figure 7.  SCF Spreadsheet Import Module with Site Selected

To save the changes to the module choose "Save and Exit" from the "File" menu. To exit the module without saving any changes choose "Exit" from the "File" menu.
