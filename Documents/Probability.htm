Probability of Exceedence Viewer Help

Probability of Exceedence is a curve that shows the probability of a particular measure (risk, concentration, ...) will occur given the criteria selections made by the user.

Menu Bar:
  The menu bar spans the top portion of the viewer window and provides the user with several standard options

  • Chart: Opens Excel, creating workbooks and charts.
  • Print: Saves a Tagged Image Format (*.tif) file with a snapshot of the dataset.
  • Save: Saves a Tab Delimited (*.txt) file with the values from the dataset.
  • Help: Contains the 'About' and 'How To...' information. The 'About' selection displays the description file of the module.

Number of Intervals:
  Choose the number of intervals between the starting and ending time points.

De-Minimus:
  The user can choose one of two criteria for De-Minimus values, which specifies how to handle leading and trailing zeroes.

  • All values: no exclusions
  • Non-zero Values: excludes all but one leading zero and excludes all zeroes after the last non-zero value. This option may modify start and end times.

The Kolmogorov-Smirnov (K-S) Goodness of Fit Test:
  KS test is used to define confidence intervals surrounding the output results produced by WEAP. The K-S test determines confidence intervals for unknown distribution functions. The probability of an event is obtained from the simulation results by identifying the probability of exceeding the event, thereby defining the distribution function of that event. In the case of time, it is the probability of exceeding a given duration of exposure. The lower and upper limits associated with the empirical distribution are defined as follows:

  • Lower Limit = Fn(x) - d
  • Upper Limit = Fn(x) + d
where d is the critical value for the Kolmogorov-Smirnov Test Statistic, which can be obtained from published tables. The following Confidence Interval pairs are provided as part of the output: 99% & 1%, 95% & 5%, 90% & 10%, 80% & 20%.

First and Second Probabilty of Exceedence Bounds:
  These selections are used to define confidence intervals surrounding the output results produced by the Sensitivity/Uncertainty model. They will determine the probabily of exceeding constituent concentations defined in the Sensitivity/Uncertainty model.

Benchmarks File:
  This gives the user the ability to specify a TREECSBenchmarkInfo XML file containing pathway benchmarks for constituents. If applicable values are found in the file they will appear as horizontal lines on the chart. Values may be omitted from the chart if they exceed the maximum value of the concentration curve and/or confidence intervals by more than 10%.

Number of Charts:
  Allows user to make more than one chart at a time, a maximum of 25 charts are possible.

Report Details:
  The user can selection a value for each criteria. Some or all of the criteria selections may be present depending on the file type being viewed.
  Note: Criteria selections are to be made in descending order as each selection may affect the criteria selections after it.

  • Dataset:   User can choose the dataset.
  • Location:   User can choose the location.
  • Age:   User can choose the age group.
  • Constituent:   User can choose a single constituent, 'All Chemicals', 'All Radionuclides' or 'All Constituents'.
  • Measure:   User can choose the measure depending upon what is applicable to the file type.
  • Unit:   User can choose the unit depending upon what is applicable to the file type.
  • Route:   User can choose a single route of exposure or 'All'.
  • Pathway:   User can choose a single pathway of exposure or 'All'.
  • Organ:   User can choose a single organ or 'All'.
