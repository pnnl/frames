5.19 INDOOR INHALATION FROM VOLATILIZATION

    The two indoor inhalation pathways for volatilization from domestic water are shower inhalation and indoor inhalation. Only one of these pathways is used for any exposure scenario. Default values for parameters for these models are given in Table 5.18.

    The default inhalation rate is set equal to the average daily inhalation rate.
