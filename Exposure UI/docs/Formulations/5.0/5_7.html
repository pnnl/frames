5.7 OTHER VEGETABLES


    This exposure pathway is similar to the leafy vegetable ingestion exposure pathway and uses similar parameters. The default parameter values for the other vegetable ingestion pathway are given in Table 5.7.

    Note that the areal soil density and the weathering constant values are the same as defined for the leafy vegetable ingestion pathway. One value is defined for each of these parameters, and the value is used for all agricultural product routes.
